## The Market Map

The AI agent landscape in early 2026 has four tiers. Kaivoo sits in a gap between Tier 2 and Tier 3 that nobody else occupies.

### Tier 1: Free Chat ($0)
| Product | What You Get | What You Don't Get |
|---|---|---|
| ChatGPT Free | GPT-5.2 Instant, ~10 msgs/5hrs | No agents, no memory persistence, no orchestration |
| Claude Free | Limited Sonnet access | No agent workflows, no persistent context |
| Gemini Free | Basic Gemini in Search/Android | No autonomous capabilities |
| Copilot Free | Edge/Bing integration | No custom agents, no local data |

**Verdict:** Chat toys. Useful for quick questions, useless for getting real work done autonomously.

### Tier 2: Consumer Paid ($15-25/mo)
| Product | Price | What You Get | The Catch |
|---|---|---|---|
| ChatGPT Plus | $20/mo | GPT-5 access, image gen, memory | Memory is a black box. No orchestration. They own your data. |
| Claude Pro | $20/mo | Opus access, projects, some memory | Limited agent features. Cloud-only. |
| Gemini AI Pro | $19.99/mo | Advanced Gemini, some agentic search | Google ecosystem lock-in. |
| Copilot Pro | $20/mo | Enhanced M365 integration | Microsoft ecosystem only. |

**Verdict:** Better chat. Some emerging agent features. But still single-AI, cloud-locked, opaque memory, no multi-agent orchestration. And $20/month *each* — most serious users are stacking 2-3 of these = $40-60/month.

### ← THE GAP: Consumer-Grade AI Agent Orchestration →

**Nobody occupies this space.** A product that gives non-technical users:
- Multi-agent workflows (not just chatting with one AI)
- Persistent, readable memory (not a black box)
- Autonomous task execution (not just answering questions)
- Data ownership (local-first, BYO keys)
- At a price point under $30/month

**This is where Kaivoo lives.** $49 one-time (desktop) / $8-12/mo (web).

### Tier 3: Developer/Power User ($100-200+/mo)
| Product | Price | What You Get | The Barrier |
|---|---|---|---|
| Claude Max | $100-200/mo | Full Opus, Claude Code, heavy agent usage | Requires developer skills. Price barrier. |
| ChatGPT Pro | $200/mo | Unlimited GPT-5, priority access | Just more chat, not orchestration. $2,400/year. |
| OpenClaw (self-hosted) | $6-100+/mo hosting | Full autonomous AI agent | Requires VPS, Docker, API keys, debugging. Developer-only. |
| Cursor / Windsurf | $15-40/mo | AI coding agents | Developer tools. Not for non-technical users. |
| Devin | $20-500/mo | Autonomous coding agent | Purely for software engineering. |

**Verdict:** Real agent capabilities exist here, but behind price walls ($100-200/mo) or skill walls (must be a developer). The average person can't access this.

### Tier 4: Enterprise ($300+/mo per seat)
| Product | Price | What You Get |
|---|---|---|
| Zapier Agents + Make | $33-300+/mo | Business workflow automation |
| Lindy AI | $20-300/mo | "AI employee" for business ops |

**Verdict:** Business tools for business people. Not consumer products.

## Competitive Moats

What Kaivoo has that competitors would struggle to replicate:

1. **"Built by its own agent system"** — No competitor can claim this. The git history proves it. This is the ultimate product-market proof: the tool built itself across 33+ sprints.

2. **Readable AI memory** — ChatGPT, Claude, and Gemini all have opaque memory. Users can't see what their AI "knows" about them. Kaivoo's Neuron Memory is readable markdown. This is a trust differentiator that big companies won't adopt because it exposes their systems.

3. **Local-first + BYO keys** — The big players will never offer this. Their business model requires cloud dependency and model lock-in. Kaivoo's model is the opposite: your machine, your choice of AI, zero ongoing cost per user.

4. **Safety Layer with transparency** — "Show your work" for AI agents. Users see thinking steps, approve destructive actions, and can audit everything. This directly addresses the #1 consumer concern about AI agents (76% verify before trusting AI decisions).

5. **One-time pricing** — In a world of $20/mo subscriptions that stack up, $49 once is psychologically powerful. It's the Obsidian model — but with AI agent capabilities that Obsidian can't match.
